# Sales Report Automation 📊

สคริปต์ Python สำหรับสรุปยอดขายจากไฟล์ Excel อัตโนมัติ

## ทำอะไรได้บ้าง
- คำนวณยอดขายรวมแยกตามสินค้า
- หาสินค้าที่ขายดีที่สุด
- หาวันที่ขายดีที่สุด
- Export รายงานเป็น Excel อัตโนมัติ 4 Sheet

## วิธีใช้งาน
1. วางไฟล์ Excel ของคุณในโฟลเดอร์เดียวกับ sales_report.py
2. แก้ชื่อไฟล์ในโค้ดให้ตรงกับไฟล์ของคุณ
3. รันคำสั่ง python sales_report.py
4. เปิดไฟล์ shop_report.xlsx ดูผลลัพธ์ได้เลย

## ติดตั้ง Library
pip install pandas

## ผลลัพธ์ที่ได้
| Sheet | รายละเอียด |
|---|---|
| Orders | ข้อมูลทั้งหมด |
| Monthly Report | ยอดขายแยกตามสินค้า |
| ยอดขายในแต่ละวัน | ยอดขายแยกตามวันที่ |
| วันที่ขายดีที่สุด | วันที่มียอดขายสูงสุด |
